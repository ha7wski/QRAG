# plans/

Tracking of what has been built and what remains to finish the app.

- [`STATUS.md`](STATUS.md) — what is **implemented** today (Steps 1–8), with the exact
  modules, endpoints, and data artifacts that exist.
- [`ROADMAP.md`](ROADMAP.md) — what is **left to do** to reach a complete product, grouped by
  priority (gaps to harden the MVP → unbuilt features F3–F6 → infra/quality).

Source-of-truth design docs: [`../project_summary.md`](../project_summary.md) (vision, feature
set F1–F6) and [`../architecture.md`](../architecture.md) (technical design). Those describe the
*target*; this folder records the *delta* against the actual code.
